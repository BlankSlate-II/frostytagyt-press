Media Inbox Setup

Folder created: Inbox → FrostyTagYT Media Replies

Auto-rules could not be saved from script (Outlook blocked it). One manual rule takes about 60 seconds and then all outlet replies go to that folder automatically.

Create the rule (60 seconds)

  1. Open Outlook → File → Manage Rules & Alerts
  2. Click New Rule → Apply rule on messages I receive → Next
  3. Check with specific words in the sender's address → in the box below click the link and add:
    roadtovr.com
    uploadvr.com
    wonderhowto.com
    reality.news
  4. Also check from people or public group → add these emails:
    tips@roadtovr.com
    contact@roadtovr.com
    dheaney@uploadvr.com
    tips@uploadvr.com
    contact@wonderhowto.com
  5. Next → check move it to the specified folder → pick FrostyTagYT Media Replies
  6. Name the rule: Emails From Media Outlets → Finish

After that, any reply from Road to VR, UploadVR, or Virtual Reality News lands in FrostyTagYT Media Replies automatically.

Or use Outlook on the web

  1. Open outlook.live.com mail rules
  2. Add rule → When email arrives from roadtovr.com or uploadvr.com or wonderhowto.com → Move to FrostyTagYT Media Replies
